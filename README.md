# Detec-o-de-Fraude-com-PySpark
Projeto de análise de transações financeiras com foco em identificação de padrões suspeitos, utilizando PySpark para processamento distribuído, com desenvolvimento de ML

Etapas:

Geração de dados sintéticos
Engenharia de dados
Regras de fraude
Score de risco
Machine Learning

Após aplicar regras de negócio, evoluí o projeto para um modelo de machine learning, utilizando regressão logística para prever probabilidade de fraude com base em variáveis como valor de transação, frequência e localização.
